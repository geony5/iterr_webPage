‘뭉게뭉게 독서록’(이하 “앱”)은 사용자의 개인정보를 중요시하며, 「개인정보 보호법」을 준수하고 있습니다.

본 개인정보 처리방침은 사용자가 제공하는 개인정보가 어떠한 용도와 방식으로 이용되고 있으며, 개인정보보호를 위해 어떠한 조치가 취해지고 있는지를 알려드립니다.

---

### **제1조 (개인정보의 처리 목적)**

앱은 별도의 회원가입 절차가 없으며, 사용자를 직접 식별할 수 있는 개인정보를 수집하지 않습니다.

다만, 서비스의 원활한 제공 및 품질 개선을 위하여 다음과 같은 목적으로 최소한의 정보를 처리합니다.

1. **데이터 백업 및 동기화:**
    
    사용자가 기록한 독서 데이터를 사용자의 개인 iCloud 계정을 통해 백업하고 여러 기기에서 동기화합니다.
    
2. **도서 정보 검색:**
    
    ISBN 바코드 스캔 또는 텍스트 검색 시, 도서 정보 조회를 위해 필요한 정보를 관련 API 서버로 전송합니다.
    
3. **부가 기능 제공:**
    
    카메라를 이용한 ISBN 바코드 인식, 사진 라이브러리를 이용한 OCR 텍스트 추출 등 사용자가 요청하는 기능을 제공합니다.
    
4. **서비스 품질 개선 및 통계 분석(Firebase Analytics):**
    
    앱의 오류 분석, 성능 개선 및 기능 향상을 위해 Firebase Analytics를 통해 비식별화된 형태의 이용 데이터를 수집·분석합니다.
    

---

### **제2조 (처리하는 개인정보의 항목)**

앱은 서비스 이용 과정에서 다음의 정보를 처리할 수 있습니다.

1. **사용자 생성 콘텐츠:**
    
    사용자가 앱 내에 직접 기록하는 모든 데이터(책 정보, 노트, 구름 분류 등).
    
    → 이 정보는 사용자의 기기 및 개인 iCloud 계정에만 저장됩니다.
    
2. **도서 검색 정보:**
    
    도서 정보 검색 시 사용되는 ISBN 코드 및 검색어 (개인 식별 불가한 정보)
    
3. **기기 접근 권한:**
    - 카메라 접근 (ISBN 바코드 인식용)
    - 사진 라이브러리 접근 (OCR 기능용)
        
        앱은 사용자의 명시적인 동의가 있는 경우에만 접근하며, 해당 정보를 서버로 전송하지 않습니다.
        
4. **자동 수집 정보 (Firebase Analytics):**
    - 앱 사용 패턴(방문한 화면, 클릭한 버튼 등)
    - 기기 정보(모델명, OS 버전, 언어 설정 등)
    - 오류 로그 및 성능 데이터
        
        ※ 이 정보는 비식별화된 형태로 수집되며, 개인을 식별하지 않습니다.

### **제3조 (개인정보의 처리 및 보유 기간)**

앱은 사용자의 개인정보를 개발자의 서버에 직접 저장하지 않으며, 다음과 같이 처리합니다.

1. 사용자가 앱 내에 기록한 데이터는 사용자의 단말기 및 iCloud 계정에 저장됩니다.
    
    → 사용자가 앱 내에서 삭제하거나 앱을 삭제하면 해당 데이터는 함께 삭제됩니다.
    
2. Firebase Analytics를 통해 수집된 데이터는 Google의 서버에 저장되며,(자세한 내용은 [Google 개인정보처리방침](https://policies.google.com/privacy) 참조)
    
    **Google의 데이터 보관 정책**에 따라 일정 기간 후 자동 삭제됩니다.

### **제6조 (개인정보의 국외 이전)**

앱은 서비스 제공을 위해 다음과 같이 개인정보를 국외로 이전하고 있습니다.

1. **iCloud 데이터 동기화**
    - **이전받는 자:** Apple Inc.
    - **이전되는 개인정보 항목:** 사용자가 앱에 기록한 모든 콘텐츠 데이터
    - **이전 국가:** 미국 (Apple iCloud 서버 소재지)
    - **이전 방법:** 사용자가 iCloud 동기화 기능 활성화 시 네트워크를 통해 실시간 전송
    - **보유·이용 기간:** 사용자가 삭제하거나 앱을 삭제할 때까지
    - **거부 방법:** iCloud 동기화 기능 비활성화 (단, 백업·동기화 기능 제한 발생)
2. **Firebase Analytics 데이터 수집**
    - **이전받는 자:** Google LLC
    - **이전되는 항목:** 앱 사용 로그, 기기 정보, 오류 데이터 등 비식별화된 정보
    - **이전 국가:** 미국 (Google Firebase 서버 소재지)
    - **이전 목적:** 서비스 품질 개선, 오류 분석 및 통계
    - **보유·이용 기간:** Google의 Firebase 데이터 보관 정책에 따름
    - **거부 방법:** 사용자는 단말기 설정에서 광고 맞춤설정 제한(Ad Personalization Off) 또는 Firebase 추적 제한 기능을 설정할 수 있습니다.
3. **도서 정보 API 이용**
    - **이전받는 자:** 네이버 클라우드 플랫폼, Google Books API
    - **이전되는 항목:** ISBN 코드, 검색어
    - **이전 목적:** 도서 정보 검색 및 결과 제공
    - **보유·이용 기간:** 검색 처리 완료 후 즉시 삭제
